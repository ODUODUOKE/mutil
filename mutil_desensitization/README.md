## 正则表达式

示例手机号： 17712340000

正则表达式：(\d{3})\d*(\d{4})
- 在正则表达式中，`括号`的作用是`分组`，提供`引用`

替换文本：$1****$2
- $1 代表第一个`()`的分组
- $2 代表第二个`()`的分组
